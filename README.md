# Ear Acupoints Educational App (Auricular Map)

Interactive auricular (ear) point map for **education / demonstration only**.

## Features
- Click points on the ear to highlight and show related organ/system notes
- Click list items (search + category filters)
- 中文 / English toggle
- Works on Vercel (Next.js App Router)

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Deploy
Push to GitHub, import into Vercel, deploy as a standard Next.js project.

## Notes
Ear maps vary across systems/schools; this app is a **teaching visualization**.
